# Portfolio

SvelteKit portfolio with a dark cyberpunk theme (black + red accents), Skeleton UI, and Tailwind.

## Setup

With [Bun](https://bun.sh):

```bash
bun install
bun run dev
```

With npm:

```bash
npm install
npm run dev
```

## Stack

- **SvelteKit** – Framework
- **Skeleton UI v2** – Components and theme system
- **Tailwind CSS v3** – Styling
- **Fonts** – Orbitron (headings), JetBrains Mono (body)

Theme and layout are in `cyberpunk-theme.ts`, `tailwind.config.js`, and `src/app.css`.
